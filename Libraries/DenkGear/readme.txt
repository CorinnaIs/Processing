DenkGear Library for Processing
Michael Zoellner, 2012

Reads and parses data from Neurosky Mindwave and ThinkGear hardware via ThinkGear Connector socket server.
Parsing the raw serial data stream from the device is not implemented yet.

Installation:
1. Download and copy the library into your Processing library folder
2. Download and install ThinkGear Connector (http://store.neurosky.com/products/thinkgear-connector)
3. Turn on your device, run ThinkGear Connector and start a sketch in Processing
